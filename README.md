# AI BTC Prediction

This project aims to predict BTC prices on a chosen date.

Currently the dataset is not updated daily.

The model is not perfect for satisfactory use.

Currently, we rely on this [Kaggle dataset](https://www.kaggle.com/datasets/novandraanugrah/bitcoin-historical-datasets-2018-2024?select=btc_1d_data_2018_to_2025.csv) which is updated every 4 hours.

## Disclamer
This project was developed to learn about Machine Learning.

Note this is still __prediction values__ and not actual values, just like weather.
